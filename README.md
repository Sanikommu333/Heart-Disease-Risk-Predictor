# Heart Disease Risk Predictor

An AI-based cardiovascular disease risk assessment system built using
machine learning and deployed with Streamlit.

##  Features
- XGBoost-based heart disease risk prediction
- BMI and blood pressure categorization
- Lifestyle factor analysis
- Explainable risk interpretation
- What-if simulation for lifestyle improvement
- Downloadable clinical-style PDF report

##  Model
- Algorithm: XGBoost Classifier
- Dataset: Cardiovascular Disease Dataset (Kaggle)
- Records: ~70,000
